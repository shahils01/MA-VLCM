=== RWARE EXAMPLE PROMPT ===

You are an expert vision language critic model for multi-agent teams able to critize given trajectories of data for their n-step returns, thus critizing the policy. This is a robotic warehouse environment with 2 agents (hard difficulty, config: tiny-2ag-hard). The reward is the sum of the minimum euclidean distance between an agent and its closest box, plus if an agent successfully places a box in the goal location (+5), and a penalty if the agents come within 3m of one another (-1). Assess the quality of the current policy based on these observations: Timestep: 1. Requested boxes: [19, 13, 8, 10]. Agent 1: position [0, 3], facing EAST, action FORWARD, carrying no. Agent 2: position [1, 7], facing WEST, action LEFT, carrying no.


=== OFFROAD EXAMPLE PROMPT ===

You are an expert vision language critic model for multi-agent teams able to critize given trajectories of data for their n-step returns, thus critizing the policy. This is an offroad navigation environment with 3 agents traversing rough terrain. The reward is based on the progress towards the goal, a heading alignment reward, minus penalties for the distance to goal, each step taken (which increases over time), idling, control effort, and a terrain traversability penalty equal to 2.0 * (1 - traversability)^2. Assess the quality of the current policy based on these observations: Timestep: 42. Agent 0 (red): position (2.30, 4.50), heading 0.50 rad, speed 1.02 m/s, dist_to_goal 15.30m, traversability 0.80, reached: no, collision: no. Agent 1 (blue): position (6.10, 8.20), heading -0.20 rad, speed 0.81 m/s, dist_to_goal 8.10m, traversability 0.90, reached: no, collision: no. Agent 2 (green): position (12.00, 1.50), heading 1.20 rad, speed 0.00 m/s, dist_to_goal 2.00m, traversability 1.00, reached: yes, collision: no.
